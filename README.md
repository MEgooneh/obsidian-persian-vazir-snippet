# Obsidian Persian Vazir Snippet

A minimal Obsidian CSS snippet that makes Persian text in Obsidian use the Vazir/Vazirmatn font.

The font is embedded in the CSS file, so it works locally without third-party font URLs.

## Install

1. Download `persian-vazirmatn.css`.
2. Copy it into your vault:

```text
YourVault/.obsidian/snippets/persian-vazirmatn.css
```

3. In Obsidian, open `Settings -> Appearance -> CSS snippets`.
4. Click reload, then enable `persian-vazirmatn`.
